
Sistema web para consulta de empresas de energia solar nos estados de GO e DF. 
 
 Funcionalidades:
-  Busca por nome da empresa 
-  Filtro por estado (DF/GO) 
-  Interface responsiva 
-  Carregamento assíncrono de dados 
 
 Tecnologias:
- HTML5, CSS3, JavaScript
- JSON para armazenamento de dados 
- Git para versionamento 
 
Instalação: 
git clone https://github.com/seu-usuario/projeto-solar.git 
cd projeto-solar 
# Abra index.html no navegador
